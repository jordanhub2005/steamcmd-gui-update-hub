## Overview

steamcmd gui hub is designed for users who want a simpler way to operate Counter-Strike 2 dedicated servers on Windows. Rather than moving between command-line steps and separate tools, it wraps SteamCMD-based setup, server control, and routine upkeep inside a graphical interface.

The project is meant for server administrators and community operators who need a straightforward place to install servers, keep them updated, manage plugins, and run RCON actions. That makes everyday CS2 server maintenance more organized and less fragmented.
